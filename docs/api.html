Authentication #

Reward Geni uses a two-step token-based authentication flow:

  1. Step 1: Call POST /auth/token with x-client-id + x-client-secret headers to get a Bearer token.
  2. Step 2: Use the token in Authorization: Bearer <token> header + x-rg-secret header for all API requests.

Reward Geni caches the token in Redis and reuses it until expiry. A new token is generated automatically when expired.

Header Required Value
Authorization Yes Bearer <accessToken>
x-rg-secret Yes <your hybridRgSecret>
Content-Type Yes application/json

POST /auth/token #

POST /auth/token

Reward Geni calls this endpoint to obtain a Bearer token before making any authenticated API requests. The token is cached in Redis with TTL matching expiresIn.

Request Headers

Credentials are passed as HTTP headers, not in the request body:

Header Required Value
Content-Type Yes application/json
x-client-id Yes Your hybridClientId
x-client-secret Yes Your hybridClientSecret

Code Examples

cURL
curl -X POST "https://your-api.example.com/auth/token" \
    -H "Content-Type: application/json" \
    -H "x-client-id: your-hybrid-client-id" \
    -H "x-client-secret: your-hybrid-client-secret"
JavaScript
const response = await fetch('https://your-api.example.com/auth/token', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'x-client-id': 'your-hybrid-client-id',
        'x-client-secret': 'your-hybrid-client-secret'
    }
});

const data = await response.json();
console.log(data.accessToken);
Node.js
const axios = require('axios');

const { data } = await axios.post(
    'https://your-api.example.com/auth/token',
    {},
    {
        headers: {
            'Content-Type': 'application/json',
            'x-client-id': 'your-hybrid-client-id',
            'x-client-secret': 'your-hybrid-client-secret'
        }
    }
);

console.log(data.accessToken);

Success Response (200)

JSON
{
  "accessToken": "eyJhbGciOiJIUzI1NiIs...",
  "expiresIn": 3600,
  "tokenType": "Bearer"
}
Field Type Description
accessToken string JWT or opaque Bearer token
expiresIn number Token lifetime in seconds (e.g., 3600 = 1 hour)
tokenType string Always Bearer

Error Responses

400 Bad Request
{
  "success": false,
  "code": "HC_400_001",
  "message": "Missing required headers: x-client-id, x-client-secret"
}
401 Unauthorized
{
  "success": false,
  "code": "HC_401_001",
  "message": "Invalid client credentials"
}

GET /v1/points/:externalUserId #

GET /v1/points/:externalUserId

Called by Reward Geni after SSO login to fetch the user's current point balance grouped by blocks.

Code Examples

cURL
curl -X GET "https://your-api.example.com/v1/points/EMP001" \
    -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..." \
    -H "x-rg-secret: your-hybrid-rg-secret"
JavaScript
const response = await fetch('https://your-api.example.com/v1/points/EMP001', {
    method: 'GET',
    headers: {
        'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIs...',
        'x-rg-secret': 'your-hybrid-rg-secret'
    }
});

const data = await response.json();
console.log(data);
Node.js
const axios = require('axios');

const { data } = await axios.get(
    'https://your-api.example.com/v1/points/EMP001',
    {
        headers: {
            'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIs...',
            'x-rg-secret': 'your-hybrid-rg-secret'
        }
    }
);

console.log(data);
